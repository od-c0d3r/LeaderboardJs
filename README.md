## Getting Started

To get a local copy up and running follow these simple example steps

 - from your terminal enter in sequence 
   - `git clone https://github.com/od-c0d3r/LeaderboardJs.git`
   - `code LeaderboardJs`
   - `npm i`
   - `npm start`
